# Products API

Rest API для товаров и категорий на NestJS

# Технологии

- NestJS
- TypeORM
- PostgreSQL
- Swagger

# Установка

1. Клонировать репозиторий
2. Установить зависимости:
    npm install

3. Создать базу данных PostgreSQL
4. Создать файл `.env`:

```
db_host=your_host
db_port=your_port
db_user=your_user
db_pass=your_password
db_name=your_db_name
```

5. Запуск:
    npm run start:dev

6. Документация API

Swagger доступ по адресу http://localhost:3000/api
